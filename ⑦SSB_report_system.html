LINE通知　設定ガイド
係長が5分で設定できます。サーバー不要・無料。
✅ 仕組みの全体像
社長がダッシュボードで「報告を求める」ボタンを押す
↓　LINE Notify API（無料）で課長のLINEに通知
↓　課長がLINEのリンクを開いて報告フォームに入力
↓　Google スプレッドシートに自動保存
↓　社長のダッシュボードに回答が表示される
設定手順
1
LINE Notify でトークンを取得
notify-bot.line.me にアクセス → ログイン → 「マイページ」→「トークンを発行する」→ SSBグループのLINEグループを選択
https://notify-bot.line.me/ja/
2
Google スプレッドシートを作成
Googleドライブで新規スプレッドシートを作成 →「報告受信台帳」と名前をつける → シートIDをコピーしておく
3
Google Apps Script で自動化
スプレッドシートの「拡張機能」→「Apps Script」→ 以下のコードを貼り付けてデプロイ
// LINE通知 + スプレッドシート保存 function doPost(e) { const data = JSON.parse(e.postData.contents); // LINEに送信 sendLine(data.message, data.token); // シートに保存 saveToSheet(data); return ContentService.createTextOutput('ok'); }
4
ダッシュボードにトークンを設定
このHTMLファイルの先頭部分の「LINE_TOKEN」と「GAS_URL」に取得した値を入力するだけで完成
const LINE_TOKEN = "ここにトークンを貼り付け"; const GAS_URL = "GASのデプロイURL";
5
課長ごとのLINEグループに分ける（任意）
拠点別にLINEグループを作り、それぞれのトークンを取得すると、該当課長だけに通知を送れます
💡 今すぐ使えるデモモード
このプロトタイプはLINEトークン未設定でも動作します。「報告を求める」ボタンを押すとダッシュボード内で通知・受信の流れを確認できます。本番運用時にトークンを設定してください。
